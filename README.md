### Runing the pygame version 
<br />

1. Clone the repo 
```sh
git clone https://github.com/thisIsShubhamSharma/s_engine
```

2. Install requirements
```sh
pip3 install -r requirements.txt
```

3. Start the engine
```sh
python3 chess_main.py
```
